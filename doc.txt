git clone - скачать удаленный репозиторий
git push - отправить в удаленный репозиторий локальную версию 
ветки лучше создавать через **checkout**, чтобы сразу перейти на новую ветвь
git pull - скачивает версию с удаленного репозиторий и проводит слияние
pull request - команда для предложения изменений
➜ запрос на вливание изменений в репозиторий
